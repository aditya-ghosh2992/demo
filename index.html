Mystic Memory

"Alphabet Card Reveal"
It is a web-based game programmed using HTML, CSS, and JavaScript. Players are presented with a grid of alphabet cards face down. They click cards to reveal the letters, aiming to match pairs within a limited number of attempts. JavaScript controls game logic, tracking matches and attempts while CSS styles the layout and visuals. Responsive design ensures compatibility across devices. With intuitive UI and engaging gameplay, "Alphabet Card Reveal" offers a fun way to enhance memory and cognitive skills

XoXo Grid

"Single Player Tic-Tac-Toe".
It is a web-based game developed using HTML, CSS, and JavaScript. The player competes against a system opponent, aiming to align three symbols horizontally, vertically, or diagonally. JavaScript handles game logic, enabling the system to make strategic moves and providing feedback on the game's progress. CSS ensures an appealing visual layout, while HTML structures the game elements. With intuitive controls and challenging system, "Single Player Tic-Tac-Toe" offers a fun and engaging experience for players looking to test their strategic thinking and enjoy a classic game in a modern digital format.

Cobra Quest

"The Old Mobile Snake Game"
It is a nostalgic web-based recreation crafted with HTML, CSS, and JavaScript. Players navigate a snake around the grid, devouring apples while avoiding collisions with the walls and its own tail. JavaScript manages game mechanics, controlling snake movement and collision detection. CSS delivers retro-inspired visuals, mimicking the classic mobile game's aesthetic. HTML structures the game elements for seamless play. With intuitive controls and addictive gameplay, "Single Player Old Mobile Snake Game" offers a delightful journey back to the days of simple yet captivating entertainment, perfect for reliving cherished memories or for timeless gaming fun.

Rock Paper Scissors

"The Classical Game".
It is a dynamic web game created using HTML, CSS, and JavaScript. Players face off against a computer opponent, selecting rock, paper, or scissors in each round. JavaScript orchestrates gameplay mechanics, including player and computer moves, outcome determination, and score tracking. CSS stylizes the interface, offering visually appealing animations and responsive design. HTML structures the game elements for seamless interaction. With intuitive controls and strategic decision-making, "Single Player Rock Paper Scissors" provides an engaging experience suitable for quick entertainment and honing predictive skills, all within the confines of a modern browser.
